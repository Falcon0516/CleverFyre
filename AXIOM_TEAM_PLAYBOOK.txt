# AgPP v1 — Agentic Payment Protocol Specification

## Abstract
AgPP defines a 4-layer standard for trustworthy AI-to-API payments.
It specifies identity derivation, intent documentation, on-chain enforcement,
and public audit for autonomous AI agents making payments on behalf of humans.

## Layer 1 — Identity
  agent_addr = HKDF-SHA256(salt="agpp-v1", ikm=org_id+":"+role+":"+sha256(code))
  Deterministic. Tamper-detectable. No wallet file required.
  Change one byte of agent code -> address changes automatically.

## Layer 2 — Intent
  Each payment is preceded by an AgPP Intent Document (JSON):
  Fields: schema, agent_id, task_canonical, api_url, api_selection_reason,
          expected_output_schema, policy_commitment, timestamp_round, chain_id
  Intent hash -> Merkle leaf -> session Merkle root committed on Algorand.
  Selective disclosure: prove any single payment without revealing others.

## Layer 3 — Enforcement
  Six ARC-4 smart contracts implement AgPP on Algorand:
  PolicyVault, IntentRegistry, SentinelEscrow,
  PaymentDNARegistry, ConsensusVault, ReputationLedger.
  All payment enforcement is cryptographic and on-chain.
  No side channels. No admin overrides.

## Layer 4 — Audit
  Full history reconstructable from public Algorand data alone.
  Filter by note_prefix="x402:axiom:" via Algorand Indexer.
  Temporal autopsy: reconstruct any moment in history from any round.
  No AXIOM server required for audit. Chain is the source of truth.

## Reference Implementation
  AXIOM — https://github.com/[your-repo]
  Live on Algorand Testnet. Contract IDs in README.

    @abimethod()
    def get_ipfs_cid(self, tx_id_hash: Bytes) -> Bytes:
        return self.intent_cids[tx_id_hash]

import hashlib, hmac
import algosdk

def derive_agent_address(org_secret: bytes, org_id: str,
                          agent_role: str, code_path: str) -> tuple[str, str]:
    """
    Returns (private_key_b64, algorand_address).
    Same inputs always produce same address.
    Change one byte of code_path -> different address (tamper detection).
    """
    code_hash = hashlib.sha256(open(code_path, "rb").read()).hexdigest()
    ikm = f"{org_id}:{agent_role}:{code_hash}".encode()
    seed = hmac.new(org_secret, ikm, hashlib.sha256).digest()
    # algosdk deterministic keypair from 32-byte seed:
    private_key = algosdk.mnemonic.from_key(seed)
    public_key = algosdk.mnemonic.to_key(private_key)
    address = algosdk.encoding.encode_address(
        algosdk.encoding.decode_address(
            algosdk.account.address_from_private_key(
                algosdk.mnemonic.to_private_key(private_key)
            )
        )
    )
    return algosdk.mnemonic.to_private_key(private_key), address

import hashlib
from typing import List

def _h(data: bytes) -> bytes:
    return hashlib.sha256(data).digest()

class MerkleTree:
    def __init__(self, leaves: List[bytes]):
        self.leaves = [_h(l) for l in leaves]
        self.tree = self._build()

    def _build(self) -> List[List[bytes]]:
        layer = self.leaves[:]
        tree = [layer]
        while len(layer) > 1:
            if len(layer) % 2 == 1:
                layer = layer + [layer[-1]]  # pad odd layer
            layer = [_h(layer[i] + layer[i+1]) for i in range(0, len(layer), 2)]
            tree.append(layer)
        return tree

    def get_root(self) -> bytes:
        return self.tree[-1][0] if self.tree and self.tree[-1] else b""

    def get_proof(self, index: int) -> List[bytes]:
        proof = []
        for layer in self.tree[:-1]:
            sibling = index ^ 1
            if sibling < len(layer):
                proof.append(layer[sibling])
            index //= 2
        return proof

    def verify(self, leaf: bytes, proof: List[bytes], root: bytes) -> bool:
        node = _h(leaf)
        for sibling in proof:
            node = _h(node + sibling)
        return node == root

"""
32-dimensional behavioral fingerprint for each agent.
Stored as int8[32] in PaymentDNARegistry box storage.
"""

import numpy as np
from scipy.spatial.distance import cosine
from scipy.stats import wasserstein_distance

class BehavioralDNA:
    def __init__(self, vector: np.ndarray = None):
        self.vector = vector if vector is not None else np.zeros(32)

    def update(self, observation: dict) -> None:
        """
        observation = {
          "amount": float,            # log-normalized -> d[0]
          "calls_per_hour": float,    # EMA -> d[1]
          "api_domains": list[str],   # Shannon entropy -> d[2]
          "sla_passed": bool,         # ratio -> d[3]
          "refunded": bool,           # ratio -> d[4]
          "category_counts": dict     # 26 buckets -> d[6..31]
        }
        """
        self.vector[0] = np.clip(np.log1p(observation.get("amount", 0)) / 10, -1, 1)
        alpha = 0.1
        self.vector[1] = (alpha * observation.get("calls_per_hour", 0) +
                          (1 - alpha) * self.vector[1])
        domains = observation.get("api_domains", [])
        if domains:
            from collections import Counter
            counts = np.array(list(Counter(domains).values()), dtype=float)
            probs = counts / counts.sum()
            self.vector[2] = float(-np.sum(probs * np.log2(probs + 1e-10)) /
                                    np.log2(len(probs) + 1))
        cat = observation.get("category_counts", {})
        total = sum(cat.values()) or 1
        for i, (_, v) in enumerate(list(cat.items())[:26]):
            self.vector[6 + i] = v / total
        self.vector = np.clip(self.vector, -1, 1)

    def drift_score(self, observation_vec: np.ndarray) -> float:
        """Cosine distance 0-1. >0.3 triggers anomaly check."""
        if np.all(self.vector == 0):
            return 0.0
        return float(cosine(self.vector, observation_vec))

    def mission_drift_score(self, expected: dict, actual: dict) -> float:
        """Wasserstein distance on category distribution. >0.4 = mission drift."""
        cats = list(set(list(expected.keys()) + list(actual.keys())))
        e = np.array([expected.get(c, 0) for c in cats], dtype=float)
        a = np.array([actual.get(c, 0) for c in cats], dtype=float)
        e /= (e.sum() or 1)
        a /= (a.sum() or 1)
        return float(wasserstein_distance(e, a))

    def to_bytes(self) -> bytes:
        return (self.vector * 127).astype(np.int8).tobytes()

    @classmethod
    def from_bytes(cls, b: bytes) -> "BehavioralDNA":
        return cls(np.frombuffer(b, dtype=np.int8).astype(float) / 127.0)

SEMANTIC_CATEGORIES = [

    ("financial_data", "stock price market index fund financial equity"),
    ("news",           "news article headline current events journalism"),
    ("search",         "web search query information lookup retrieval"),

def get_category_embeddings() -> dict:
    global _cat_embs
    if _cat_embs is None:
        m = get_model()
        _cat_embs = {label: m.encode(desc, normalize_embeddings=True)
                     for label, desc in SEMANTIC_CATEGORIES}
    return _cat_embs

def route_api(api_description: str,
               budget_map: dict) -> tuple[Optional[str], float]:
    """
    Returns (matched_category, score).
    Returns (None, score) if score < 0.7 — BLOCK this payment.
    """
    emb = embed(api_description)
    best_cat, best_score = None, 0.0
    for cat, vec in get_category_embeddings().items():
        s = float(np.dot(emb, vec))
        if s > best_score:
            best_score, best_cat = s, cat
    if best_score < 0.7 or best_cat not in budget_map:
        return None, best_score
    return best_cat, best_score

from sklearn.ensemble import IsolationForest
import numpy as np
from collections import deque
import time

class AnomalyDetector:
    def __init__(self, window: int = 50):
        self.window = window
        self.history: deque = deque(maxlen=window)
        self.timestamps: deque = deque(maxlen=200)
        self.model = None

    def record(self, features: list) -> None:
        self.history.append(features)
        self.timestamps.append(time.time())
        if len(self.history) >= 10:
            X = np.array(self.history)
            self.model = IsolationForest(contamination=0.1, random_state=42)
            self.model.fit(X)

    def is_anomaly(self, features: list) -> bool:
        if self.model is None:
            return False
        return self.model.predict([features])[0] == -1

    def burst_check(self, window_sec: float = 30.0, max_calls: int = 20) -> bool:
        """Returns True if burst rate exceeds max_calls in window_sec."""
        now = time.time()
        recent = sum(1 for t in self.timestamps if now - t < window_sec)
        return recent > max_calls

import os, base64
from algosdk.v2client import indexer as idx_client
from dataclasses import dataclass, field
from typing import Dict, List

AXIOM_NOTE_PREFIX = base64.b64encode(b"x402:axiom:").decode()

@dataclass
class AgentSnapshot:
    address: str
    reputation_score: int = 500
    tier: int = 2
    dna_drift: float = 0.0
    policy_status: str = "active"
    payments_made: int = 0
    payments_blocked: int = 0

@dataclass
class SystemSnapshot:
    round: int = 0
    agents: Dict[str, AgentSnapshot] = field(default_factory=dict)
    events: List[dict] = field(default_factory=list)

    def apply(self, note: str, tx: dict) -> None:
        sender = tx.get("sender", "unknown")
        if sender not in self.agents:
            self.agents[sender] = AgentSnapshot(address=sender)
        agent = self.agents[sender]
        if "BLOCK" in note or "QUARANTINE" in note:
            agent.payments_blocked += 1
        elif "x402:axiom:" in note:
            agent.payments_made += 1
        self.events.append({
            "type": "PAYMENT" if "BLOCK" not in note else "BLOCKED",
            "sender": sender,
            "round": tx.get("confirmed-round", 0),
            "note": note,
        })

class TemporalQuery:
    def __init__(self):
        self.indexer = idx_client.IndexerClient(
            "",
            os.getenv("INDEXER_SERVER", "http://localhost"),
            int(os.getenv("INDEXER_PORT", "8980"))
        )

    def reconstruct_at(self, round_number: int) -> SystemSnapshot:
        all_txns, next_token = [], None
        while True:
            kwargs = dict(max_round=round_number, note_prefix=AXIOM_NOTE_PREFIX)
            if next_token:
                kwargs["next_page"] = next_token
            try:
                result = self.indexer.search_transactions(**kwargs)
            except Exception:
                break
            all_txns.extend(result.get("transactions", []))
            next_token = result.get("next-token")
            if not next_token:
                break
        snap = SystemSnapshot(round=round_number)
        for tx in sorted(all_txns, key=lambda t: t.get("confirmed-round", 0)):
            raw_note = tx.get("note", "")
            try:
                note = base64.b64decode(raw_note).decode(errors="replace")
            except Exception:
                note = ""
            snap.apply(note, tx)
        return snap

        cap = self.policy.get("spend_cap_algo", 1.0)
        blocked_at = None
        for i in range(20):
            amount = (cap / 10) * (1.5 ** i)
            if amount > cap:
                blocked_at = i
                break
        return AttackResult(
            id="ATK-001", name="Gradual Escalation",
            succeeded=(blocked_at is None),
            details=f"Blocked at call {blocked_at}" if blocked_at else "Not blocked",
            recommendation="Decrease spend_cap_algo or add cumulative rate limit"
                           if blocked_at is None else ""
        )

    def _atk_002_domain_spoof(self) -> AttackResult:
        from axiom_agpp.semantic import route_api
        spoofed = "weather temperature forecast data premium API endpoint"
        cat, score = route_api(spoofed, self.policy.get("budget_map", {}))
        vulnerable = cat is not None and score > 0.7
        return AttackResult(
            id="ATK-002", name="Domain Spoof",
            succeeded=vulnerable,
            details=f"Matched: {cat} score={score:.2f}",
            recommendation="Tighten category descriptions in policy.yaml" if vulnerable else ""
        )

    def _atk_003_burst(self) -> AttackResult:
        burst_max = self.policy.get("burst_max_calls", 20)
        vulnerable = 100 > burst_max
        return AttackResult(
            id="ATK-003", name="Burst Attack (100 micro-payments)",
            succeeded=vulnerable,
            details=f"burst_max_calls={burst_max}",
            recommendation="Set burst_window_sec=10, burst_max_calls=10" if vulnerable else ""
        )

    def _atk_004_semantic_confusion(self) -> AttackResult:
        from axiom_agpp.semantic import route_api
        desc = "financial weather news market temperature forecast data analysis"
        cat, score = route_api(desc, self.policy.get("budget_map", {}))
        ambiguous = score < 0.75
        return AttackResult(
            id="ATK-004", name="Semantic Confusion",
            succeeded=not ambiguous,
            details=f"Best match: {cat} at {score:.2f}",
            recommendation="Add disambiguation terms to category descriptions"
                           if not ambiguous else ""
        )

    def _atk_005_dms_timing(self) -> AttackResult:
        window = self.policy.get("renewal_window_rounds", 1000)
        vulnerable = window > 500
        return AttackResult(
            id="ATK-005", name="Dead Man Switch Timing Attack",
            succeeded=vulnerable,
            details=f"renewal_window_rounds={window}",
            recommendation="Set renewal_window_rounds <= 500 (~30 min on Algorand)"
                           if vulnerable else ""
        )

    def _atk_006_intent_replay(self) -> AttackResult:
        has_protection = self.policy.get("replay_protection", False)
        return AttackResult(
            id="ATK-006", name="Intent Replay Attack",
            succeeded=not has_protection,
            details=f"replay_protection={has_protection}",
            recommendation="Set replay_protection: true in policy.yaml"
                           if not has_protection else ""
        )

    def print_report(self, results: List[AttackResult]) -> None:
        print("\n╔══ AXIOM RED TEAM REPORT ════════════════════════╗")
        for r in results:

            if r.recommendation:
                print(f"         RECOMMENDATION: {r.recommendation}")
        blocked = sum(1 for r in results if not r.succeeded)
        print(f"\n  {blocked}/{len(results)} attacks blocked.")
        print("╚════════════════════════════════════════════════╝\n")

import os, base64
from algosdk.v2client import indexer as idx_client

_last_round = 0
AXIOM_PREFIX = base64.b64encode(b"x402:axiom:").decode()

def _get_indexer():
    return idx_client.IndexerClient(
        "",
        os.getenv("INDEXER_SERVER", "http://localhost"),
        int(os.getenv("INDEXER_PORT", "8980"))
    )

async def poll_new_events() -> list:
    global _last_round
    idx = _get_indexer()
    try:
        result = idx.search_transactions(
            min_round=_last_round + 1,
            note_prefix=AXIOM_PREFIX
        )
        txns = result.get("transactions", [])
        if txns:
            _last_round = max(t.get("confirmed-round", 0) for t in txns)
        return [_normalize(tx) for tx in txns]
    except Exception:
        return []

def _normalize(tx: dict) -> dict:
    raw = tx.get("note", "")
    try:
        note = base64.b64decode(raw).decode(errors="replace")
    except Exception:
        note = ""
    etype = "PAYMENT"
    if "BLOCK"     in note: etype = "BLOCKED"
    elif "WARN"     in note: etype = "WARNING"
    elif "QUARANTINE" in note: etype = "QUARANTINE"
    elif "DRIFT"    in note: etype = "DRIFT"
    elif "EXPIRE"   in note: etype = "EXPIRED"
    return {
        "type":   etype,
        "tx_id":  tx.get("id", ""),
        "sender": tx.get("sender", ""),
        "round":  tx.get("confirmed-round", 0),
        "note":   note,
        "amount": tx.get("payment-transaction", {}).get("amount", 0),
        "ts":     tx.get("round-time", 0),
    }

Q: "What makes this different from a payment proxy?"
A: "A proxy records that a payment happened. AXIOM proves WHY it happened
   cryptographically via a Merkle commitment on Algorand. You can audit
   any single payment without seeing any other. Selective Merkle disclosure.
   That is a completely different problem than what any proxy solves."

Q: "How do you handle agent security?"
A: "Three layers. Deterministic identity — no wallet files, key derived at
   runtime from code hash, tamper the binary and the address changes
   automatically. Behavioral DNA — we detect drift from the agent's payment
   fingerprint, not just thresholds. Graduated Dead Man's Switch — if the
   operator loses control, payments freeze in stages within hours."

Q: "Why Algorand specifically?"
A: "Four reasons. Box storage — store DNA vectors and Merkle roots on-chain
   for under a cent per box, impossible on Ethereum without prohibitive gas.
   ARC-28 events for real-time Indexer streaming. 4-second finality for
   real-time agent payments. And AtomicTransactionComposer — our M-of-N
   consensus bundles 3 peer consent transactions plus payment atomically.
   That composability is unique to Algorand."

Q: "Could this work in production?"
A: "Yes. pip install axiom-agpp. 5 lines. The contracts are live on testnet
   with real ALGO right now. CLI is functional. We also shipped a red team
   simulator — you attack your own policy before deploying. That is not a
   hackathon mindset. That's production engineering."

Q: "Privacy concern — everything on a public chain?"
A: "Only hashes go on-chain. Intent documents go to IPFS — accessible only
   if you know the CID. Merkle selective disclosure: reveal exactly one
   payment to a regulator, nothing else. Agent task content and output:
   never on-chain. Privacy by architecture, not by policy."

Q: "Can peer agents collude in the consensus mechanism?"
A: "Agents are identified by deterministic addresses derived from code hash.
   Collusion requires compromising M of N peers with different code hashes.
   And even then: the DNA registry and Dead Man's Switch would flag the
   anomaly pattern within hours. Defense in depth."

Q: "What is AgPP and why does it matter?"
A: "AgPP is the Agentic Payment Protocol — an open specification defining
   the 4-layer standard for trustworthy AI-to-API payments. AXIOM is
   its reference implementation. Standards outlive products.
   Any developer can implement AgPP independently. We own the specification,
   not just the software. That is the moat."
