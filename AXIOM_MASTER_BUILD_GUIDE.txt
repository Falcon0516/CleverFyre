  >>>START PROMPT CA-2 — PASTE INTO ANTIGRAVITY
  Using AlgoKit Python ARC-4 (Puya compiler), create a smart contract file at:
  projects/cleverfyre/smart_contracts/policy_vault/contract.py

  The contract is called PolicyVault and is the central policy enforcement contract
  for the AXIOM agentic payment protocol. Implement the following:

  Box storage schema:
    Key: agent_addr (bytes, 32 bytes = Algorand address)
    Value: PolicyRecord struct containing:
      - expiry_round: UInt64
      - missed_renewals: UInt64
      - spend_cap_tier: UInt64  (0=full, 1=1 ALGO cap, 2=0.1 ALGO cap, 3=frozen)
      - operator_pubkey: Bytes (32 bytes)
      - renewal_seed: Bytes (24 bytes = last 3 payment hashes concatenated as 8 bytes each)

  Methods to implement:
  1. init_policy(agent_addr: Bytes, window_rounds: UInt64, operator_pubkey: Bytes) -> None
     - Creates a new PolicyRecord for an agent
     - Sets expiry_round = Global.round + window_rounds
     - Sets missed_renewals = 0, spend_cap_tier = 0
     - Emits ARC-28 event: PolicyInitialized(agent_addr, expiry_round)

  2. renew_with_proof(agent_addr: Bytes, challenge_sig: Bytes) -> None
     - Reads the PolicyRecord for agent_addr
     - Computes renewal_challenge = sha256(renewal_seed)[:8]
     - Verifies ed25519_verify(challenge_sig, renewal_challenge, operator_pubkey)
     - If valid: resets expiry_round, resets missed_renewals, updates spend_cap_tier to 0
     - Emits ARC-28 event: PolicyRenewed(agent_addr, new_expiry_round)

  3. check_and_enforce(agent_addr: Bytes, amount_microalgo: UInt64) -> UInt64
     - Reads PolicyRecord
     - If current round > expiry_round: increment missed_renewals, update tier
       Tier logic: missed_renewals==1 → tier=1, ==2 → tier=2, >=3 → tier=3
     - If tier==3: assert False, "POLICY_FROZEN"
     - If tier==1 and amount > 1_000_000: assert False, "AMOUNT_EXCEEDS_TIER1_CAP"
     - If tier==2 and amount > 100_000: assert False, "AMOUNT_EXCEEDS_TIER2_CAP"
     - Returns current tier
     - Emits ARC-28 event: PolicyChecked(agent_addr, tier, amount_microalgo)

  4. record_payment_hash(agent_addr: Bytes, payment_hash: Bytes) -> None
     - Updates renewal_seed: shift 8 bytes left, append new payment_hash[:8]
     - Only callable by SentinelEscrow contract (check caller is app_addr of escrow)

  5. get_policy_status(agent_addr: Bytes) -> Bytes
     - Returns ABI-encoded PolicyRecord for the agent

  Use proper ARC-4 decorators (@abimethod, @subroutine).
  Include proper imports from algopy.
  Add a comment at the top: # AXIOM PolicyVault - AgPP Enforcement Layer v1

  >>>START PROMPT CA-4 — PASTE INTO ANTIGRAVITY
  Using AlgoKit Python ARC-4, create:

  Contract name: SentinelEscrow
  Comment at top: # AXIOM SentinelEscrow - Payment Hub and Enforcement v1

  This is the most important contract. All ALGO payments flow through it.

  Box storage:
    Key: escrow_id (32 bytes) → EscrowRecord:
      payer: Bytes (32 bytes, Algorand address)
      provider: Bytes (32 bytes, Algorand address)
      amount: UInt64 (microALGO)
      deadline_round: UInt64
      quarantine_flag: UInt64 (0=normal, 1=quarantined)
      consensus_required: UInt64 (0=no, 1=yes)
      intent_hash: Bytes (32 bytes)
      block_reason: UInt64 (0=none, see reason codes)
      status: UInt64 (0=pending, 1=released, 2=refunded, 3=quarantined)

  Global state:
    policy_vault_app_id: UInt64
    dna_registry_app_id: UInt64
    reputation_ledger_app_id: UInt64
    admin_address: Bytes

  1. bootstrap(policy_vault_id: UInt64, dna_registry_id: UInt64, reputation_id: UInt64) -> None

     - Sets global state for cross-contract calls

  2. deposit(provider_addr: Bytes, intent_hash: Bytes, deadline_rounds: UInt64, requires_consensus: UInt64) -> Bytes
     - Receives ALGO payment (attached to transaction)
     - Generates escrow_id = sha256(intent_hash + Global.latest_timestamp.bytes)
     - Stores EscrowRecord with status=0 (pending)
     - Emits ARC-28 event: EscrowDeposited(escrow_id, payer, provider_addr, amount)
     - Returns escrow_id

  3. release(escrow_id: Bytes) -> None
     - Only callable by oracle address (admin_address) or if deadline reached
     - Sends amount to provider address
     - Sets status=1
     - Emits: EscrowReleased(escrow_id, provider, amount)
     - After release: log note "x402:axiom:release:" + escrow_id.hex()

  4. quarantine(escrow_id: Bytes, anomaly_reason: UInt64) -> None
     - Sets quarantine_flag=1, status=3, block_reason=anomaly_reason
     - Emits: EscrowQuarantined(escrow_id, anomaly_reason)
     - Note field: "x402:axiom:quarantine:" + escrow_id.hex()

  5. admin_resolve(escrow_id: Bytes, approved: UInt64) -> None
     - Only callable by admin_address
     - If approved==1: calls release(escrow_id)
     - If approved==0: calls refund(escrow_id)

  6. refund(escrow_id: Bytes) -> None
     - Sends amount back to payer
     - Sets status=2
     - Emits: EscrowRefunded(escrow_id, payer, amount)
     - Note field: "x402:axiom:refund:" + escrow_id.hex()

  7. log_blocked_payment(agent_addr: Bytes, api_url_hash: Bytes, amount: UInt64, reason_code: UInt64) -> None
     - Emits ARC-28 event: PaymentBlocked(agent_addr, api_url_hash, amount, reason_code, Global.round)
     - Note field: "x402:axiom:blocked:" + reason_code as string

  All transactions must include note field prefix "x402:axiom:" for Indexer filtering.
  >>>END PROMPT CA-4

          # Use seed as private key material
          pk_b64 = base64.b64encode(seed).decode()
          # Return derived address and privkey
          return pk_b64, encoding.encode_address(seed[:32])

      def sign(self, data: bytes) -> bytes:

  Pure Python Merkle tree implementation (no dependencies beyond hashlib):

  import hashlib
  from dataclasses import dataclass
  from typing import Optional

  class MerkleTree:
      def __init__(self, leaves: list[bytes]):
          """Initialize tree from list of raw leaf data (will be hashed internally)"""
          self.raw_leaves = leaves
          self.leaves = [self._hash(leaf) for leaf in leaves]
          self.tree = self._build_tree()

      def _hash(self, data: bytes) -> bytes:
          return hashlib.sha256(data).digest()

      def _build_tree(self) -> list[list[bytes]]:
          """Build complete Merkle tree bottom-up, returns list of levels"""
          if not self.leaves:
              return [[self._hash(b"empty")]]
          current_level = list(self.leaves)
          levels = [current_level]
          while len(current_level) > 1:
              if len(current_level) % 2 != 0:
                  current_level.append(current_level[-1])  # duplicate last
              next_level = []
              for i in range(0, len(current_level), 2):
                  combined = current_level[i] + current_level[i+1]
                  next_level.append(self._hash(combined))
              levels.append(next_level)
              current_level = next_level
          return levels

      def get_root(self) -> bytes:
          return self.tree[-1][0]

      def get_proof(self, leaf_index: int) -> list[tuple[str, bytes]]:
          """Returns list of (direction, hash) tuples. direction = 'left' or 'right'"""
          proof = []
          idx = leaf_index
          for level in self.tree[:-1]:
              if idx % 2 == 0:
                  sibling_idx = idx + 1 if idx + 1 < len(level) else idx
                  proof.append(('right', level[sibling_idx]))
              else:
                  proof.append(('left', level[idx - 1]))
              idx //= 2
          return proof

      def verify(self, leaf_data: bytes, proof: list[tuple[str, bytes]], root: bytes) -> bool:
          current = self._hash(leaf_data)
          for direction, sibling in proof:
              if direction == 'left':
                  current = self._hash(sibling + current)
              else:
                  current = self._hash(current + sibling)
          return current == root

      def get_root_hex(self) -> str:
          return self.get_root().hex()

  import numpy as np
  from dataclasses import dataclass, field
  from scipy.spatial.distance import cosine
  from scipy.stats import wasserstein_distance
  import json

  @dataclass
  class BehavioralDNA:
      vector: np.ndarray = field(default_factory=lambda: np.zeros(32, dtype=np.float32))
      total_payments: int = 0
      drift_events: int = 0
      category_distribution: dict = field(default_factory=dict)
      expected_category_distribution: dict = field(default_factory=dict)

      DRIFT_THRESHOLD = 0.3
      MISSION_DRIFT_THRESHOLD = 0.4

      def update(self, observation: dict) -> None:
          """Update DNA vector with new payment observation.
          observation keys: amount_log (float), frequency (float), domain_entropy (float),
          sla_pass (bool), category (str), category_budget_pct (float)"""
          new_obs = self._observation_to_vector(observation)
          # EMA blend: 90% old + 10% new
          alpha = 0.1
          self.vector = (1 - alpha) * self.vector + alpha * new_obs
          self.total_payments += 1
          # Update category distribution
          cat = observation.get('category', 'unknown')
          self.category_distribution[cat] = self.category_distribution.get(cat, 0) + 1

      def _observation_to_vector(self, obs: dict) -> np.ndarray:
          v = np.zeros(32, dtype=np.float32)
          v[0] = min(obs.get('amount_log', 0) / 20.0, 1.0)  # normalized log amount
          v[1] = min(obs.get('frequency', 0) / 60.0, 1.0)   # calls per hour normalized
          v[2] = min(obs.get('domain_entropy', 0) / 5.0, 1.0)
          v[3] = 1.0 if obs.get('sla_pass', True) else 0.0
          v[4] = min(obs.get('refund_rate', 0), 1.0)
          # Categories in v[6:32]
          return v

      def drift_score(self, new_observation: dict) -> float:
          """Cosine distance from current DNA to new observation. 0=no drift, 1=max drift"""
          if self.total_payments < 5:
              return 0.0  # not enough history
          new_vec = self._observation_to_vector(new_observation)
          if np.all(self.vector == 0):
              return 0.0
          return float(cosine(self.vector, new_vec))

      def mission_drift_score(self) -> float:
          """Wasserstein distance between expected and actual category spend"""
          if not self.expected_category_distribution or not self.category_distribution:
              return 0.0
          categories = list(set(list(self.expected_category_distribution.keys()) +
                               list(self.category_distribution.keys())))
          total = max(sum(self.category_distribution.values()), 1)
          actual = [self.category_distribution.get(c, 0) / total for c in categories]
          expected_total = max(sum(self.expected_category_distribution.values()), 1)
          expected = [self.expected_category_distribution.get(c, 0) / expected_total for c in categories]
          return float(wasserstein_distance(expected, actual))

      def is_drift_detected(self, new_observation: dict) -> bool:
          return self.drift_score(new_observation) > self.DRIFT_THRESHOLD

      def is_mission_drift(self) -> bool:
          return self.mission_drift_score() > self.MISSION_DRIFT_THRESHOLD

      def to_bytes(self) -> bytes:
          """Serialize to 32 bytes for on-chain storage (int8 quantization)"""
          quantized = (self.vector * 127).clip(-128, 127).astype(np.int8)
          return quantized.tobytes()

  import numpy as np
  from sklearn.ensemble import IsolationForest
  from collections import deque
  import time

  class AnomalyDetector:
      def __init__(self, contamination: float = 0.05, window_size: int = 100):
          self.model = IsolationForest(contamination=contamination, random_state=42)
          self.history = deque(maxlen=window_size)
          self.fitted = False
          self.call_timestamps = deque(maxlen=1000)
          self.burst_window_seconds = 30
          self.burst_max_calls = 5

      def record_payment(self, amount: int, category: str, response_time_ms: int) -> None:
          now = time.time()
          self.call_timestamps.append(now)
          features = [amount, hash(category) % 1000, response_time_ms]
          self.history.append(features)
          if len(self.history) >= 20 and len(self.history) % 10 == 0:
              self.model.fit(list(self.history))
              self.fitted = True

      def is_anomaly(self, amount: int, category: str, response_time_ms: int) -> tuple[bool, float]:
          """Returns (is_anomaly, anomaly_score). Score < 0 = anomaly."""
          if not self.fitted or len(self.history) < 10:
              return False, 0.0
          features = [[amount, hash(category) % 1000, response_time_ms]]
          score = float(self.model.score_samples(features)[0])
          return score < -0.5, score

      def is_burst_attack(self) -> bool:
          """Check if recent call frequency exceeds burst threshold"""
          now = time.time()
          recent = [t for t in self.call_timestamps if now - t < self.burst_window_seconds]
          return len(recent) >= self.burst_max_calls

  from sentence_transformers import SentenceTransformer
  import numpy as np
  from dataclasses import dataclass

  @dataclass
  class SemanticCategory:
      label: str
      budget_pct: float
      keywords: list[str]
      embedding: np.ndarray = None  # computed lazily

  class SemanticRouter:
      MODEL_NAME = "all-MiniLM-L6-v2"
      SIMILARITY_THRESHOLD = 0.7

      def __init__(self):
          self.model = SentenceTransformer(self.MODEL_NAME)
          self.categories: list[SemanticCategory] = []

      def load_from_policy(self, policy_dict: dict) -> None:
          """Load categories from policy.yaml semantic_categories section"""
          for cat_def in policy_dict.get('semantic_categories', []):
              text = cat_def['label'] + " " + " ".join(cat_def.get('keywords', []))
              embedding = self.model.encode(text)
              self.categories.append(SemanticCategory(
                  label=cat_def['label'],
                  budget_pct=cat_def.get('budget_pct', 0),
                  keywords=cat_def.get('keywords', []),
                  embedding=embedding
              ))

      def route(self, api_description: str) -> tuple[str, float]:
          """Returns (category_label, similarity_score). Returns ('BLOCKED', 0.0) if below threshold."""
          if not self.categories:
              return 'UNKNOWN', 0.5
          query_embedding = self.model.encode(api_description)
          best_label = 'BLOCKED'
          best_score = 0.0
          for cat in self.categories:
              similarity = self._cosine_sim(query_embedding, cat.embedding)
              if similarity > best_score:
                  best_score = similarity
                  best_label = cat.label
          if best_score < self.SIMILARITY_THRESHOLD:
              return 'BLOCKED', best_score
          return best_label, best_score

      def _cosine_sim(self, a: np.ndarray, b: np.ndarray) -> float:
          return float(np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b) + 1e-10))

      def embed_text(self, text: str) -> np.ndarray:
          return self.model.encode(text)

                  response = wrapper_self._handle_402(client_self, request, response, **kwargs)

  export class AXIOMStream {
    private ws: WebSocket | null = null;
    private listeners: ((event: AXIOMEvent) => void)[] = [];
    private reconnectInterval = 3000;
    private useMockData = false;

    connect(): void {
      try {
        this.ws = new WebSocket(WS_URL);
        this.ws.onmessage = (msg) => {
          const event: AXIOMEvent = JSON.parse(msg.data);
          this.listeners.forEach(fn => fn(event));
        };
        this.ws.onerror = () => {
          this.useMockData = true;
          this.startMockStream();
        };
        this.ws.onclose = () => {
          setTimeout(() => this.connect(), this.reconnectInterval);
        };
      } catch {
        this.useMockData = true;
        this.startMockStream();
      }
    }

    onEvent(callback: (event: AXIOMEvent) => void): void {
      this.listeners.push(callback);
    }

  Q: "What makes this different from a payment proxy?"
  A: "A proxy records that a payment happened. AXIOM proves WHY it happened
     and cryptographically links it to the agent's reasoning via Merkle commitment
     on Algorand. Selective disclosure — prove one payment to a regulator
     without revealing any other. That's a different problem entirely."

  Q: "How do you handle agent security?"
  A: "Three layers. Deterministic identity — no wallet files, key derived from
     code hash, tamper the binary and the key changes automatically.
     Behavioral DNA — detect drift from payment fingerprint, not just thresholds.
     Graduated Dead Man's Switch — graduated freeze, not binary kill switch."

  Q: "Why Algorand specifically?"
  A: "Box storage lets us store DNA vectors and Merkle roots on-chain for $0.001
     per box — impossible on Ethereum without prohibitive gas. ARC-28 events for
     real-time Indexer streaming. 4-second finality. And AtomicTransactionComposer
     — our multi-agent consensus bundles 3 peer consent transactions + the payment
     atomically. That composability is unique to Algorand."

  Q: "Could this work in production?"
  A: "Yes. pip install axiom-agpp, 5 lines, any existing AI agent framework.
     Contracts are live on Algorand Testnet right now. We shipped a red team
     simulator — you attack your own policy before deploying. That's not a
     hackathon prototype mindset. That's production engineering."

  Q: "Privacy concern — everything is on a public chain?"
  A: "Only hashes go on-chain. Intent documents go to IPFS, accessible only if
     you know the CID. The Merkle tree enables exactly the disclosure you want
     — one payment to a regulator, nothing else. Agent task content and output:
     never on-chain. Privacy by architecture."

     Enforcement layer, Audit layer. AXIOM is its reference implementation.
     Standards outlive products. We want this to be the foundation other
     teams build on, not just a hackathon project."
